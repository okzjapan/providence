# JRA 競技構造リファレンス

## 1. クラス体系

JRA は収得賞金に基づくピラミッド構造をとる。馬は勝利を重ねてクラスを昇級していく。

| クラス | 旧名称 | 備考 |
|--------|--------|------|
| 新馬 | — | デビュー戦。過去走データなし（コールドスタート問題） |
| 未勝利 | — | 新馬で勝てなかった馬。2 歳春〜3 歳秋まで |
| 1 勝クラス | 500 万下 | |
| 2 勝クラス | 1000 万下 | |
| 3 勝クラス | 1600 万下 | |
| オープン | — | エリート層 |
| リステッド（L） | — | 準重賞。年間約 63 レース |
| 重賞 G3 | — | |
| 重賞 G2 | — | |
| 重賞 G1 | — | 最高峰。年間約 26 レース |

**AI 予測への含意**:
- 格上げ直後は苦戦しやすく、降級は好走しやすい傾向がある
- クラスの数値化（序数エンコーディング）が特徴量として有効
- 新馬戦はコールドスタート問題のため専用モデルが必要

| 阪神 | 右 | 内 357m / 外 474m | 内 1.9m / 外 2.4m | 野芝 | 坂あり。パワー型有利 |

### 3.2 ローカル場

| 競馬場 | 回り | 直線距離 | 高低差 | 芝質 | 特徴 |
|--------|------|----------|--------|------|------|
| 中京 | 左 | 412m | 3.5m | 野芝 | 2012 年改修済み |
| 新潟 | 左 | 内 359m / 外 659m | 内 0.8m / 外 2.2m | 野芝 | 外回り日本最長直線。千直は特殊コース |
| 小倉 | 右 | 293m | 3.0m | 野芝 | 小回り・先行有利 |
| 福島 | 右 | 292m | 1.9m | 野芝 | 小回りローカルの典型 |
| 札幌 | 右 | 266m | 0.7m | 洋芝 | JRA 最小高低差。パワー・スタミナ型有利 |
| 函館 | 右 | 262m | 3.5m | 洋芝 | JRA 最短直線 |

### 3.3 AI 構築で重要なポイント

- **洋芝 vs 野芝**: 札幌・函館の洋芝はクッション性が高く時計がかかる。パワー・スタミナ要求が野芝と根本的に異なる
- **直線距離**: 脚質の有利不利を決定する最大要因。短い = 先行有利、長い = 差し有利
- **左回り vs 右回り**: 左回り（東京・中京・新潟）で走れない馬もいる
- **コース替わり**: 内回り / 外回りで直線距離・高低差が異なる
- **競馬場改修**: 京都（2020-2023）改修後は馬場傾向が変化。改修前データの有効期間に注意

## 4. 芝 / ダート

JRA では芝とダートは根本的に別競技として扱う。

| 観点 | 芝 | ダート |
|------|-----|--------|
| サーフェス | 天然芝（野芝 or 洋芝） | 砂 |
| 馬場状態 | 良 / 稍重 / 重 / 不良 | 良 / 稍重 / 重 / 不良 |
| 雨の影響 | 時計がかかる（重くなる） | **高速化することがある**（逆転現象） |
| 重視能力 | 瞬発力・切れ味・末脚 | テンのスピード・先行力・パワー・持続力 |
| 脚質傾向 | 差し・追込が届きやすい | 先行が圧倒的有利 |

**AI 構築への含意**: 芝 / ダートでモデルを必ず分離する。混合モデルは性能が劣化する。

## 5. 馬場状態とトラックバイアス

### 5.1 馬場状態

含水率に基づき 4 段階で公式に判定:

- **良**: 乾燥。高速馬場
- **稍重**: やや含水。タイムがやや遅くなる
- **重**: 水分多い。パワーが要求される
- **不良**: 水浸し。タフな馬場

### 5.2 トラックバイアス

開催日数の経過で馬場状態が変化し、有利不利の偏りが生じる:

- **開催前半（1-4 日目）**: 内枠・先行馬有利（芝が綺麗）
- **開催後半（5 日目以降）**: 内側の芝が荒れ、外差し馬場に変化
- **ダート**: レース間にハロー掛け（整地）されるため芝ほどバイアスが出にくい

**特徴量**: 同一開催の経過日数（開催何日目か）がトラックバイアスの有力な代理変数。

## 6. 斤量（負担重量）

### 6.1 レース区分

| 区分 | 決定方法 | 採用レース |
|------|----------|-----------|
| 馬齢重量 | 年齢で決定。牝馬は牡馬より 2kg 減 | 2 歳限定戦・3 歳限定戦 |
| 定量 | 年齢・性別で均一 | G1 レース |
| 別定 | 基礎重量 + 収得賞金に応じた加算 | 重賞・特別競走 |
| ハンデキャップ | JRA 委員会が個別に設定。最大 10kg 差 | ハンデ戦 |

### 6.2 減量騎手

| 記号 | 条件 | 減量 |
|------|------|------|
| ▲ | 30 勝以下 | 3kg 減 |
| △ | 31-50 勝 | 2kg 減 |
| ☆ | 51-100 勝 | 1kg 減 |

### 6.3 斤量の影響

- 斤量 1kg ≒ 0.084 秒 ≒ 0.42 馬身（通説の「1kg = 1 馬身」より小さい）
- **馬体重比**（斤量 / 馬体重）が 12.5% を超えると複勝回収率が低下する傾向
- **ハンデ戦の歪み**: データ分析によると、斤量が重い馬ほど成績が良い傾向あり（ハンデが適切に機能していない）

## 7. 馬齢と成長曲線

| 年齢 | 特徴 | 単勝回収率 |
|------|------|-----------|
| 2 歳 | 成長途上。データ少。新馬戦あり | — |
| 3 歳 | 急成長期。春と秋で能力が大きく変わる | — |
| 4 歳 | 全盛期前半 | 72 |
| 5 歳 | 全盛期後半 | 77 |
| 6 歳 | 衰退開始 | 61 |
| 7 歳以上 | 衰退 | 62 |

- **3 歳夏の斤量差**: 古馬より 3-5kg 軽い = 3-5 馬身分のアドバンテージ。3 歳以上レースでの 3 歳馬は回収率が高い
- **成長曲線の位置**: 馬齢の数値だけでなく「この馬は成長途上か衰退期か」を推定する必要あり

## 8. ローテーションと休養

| 間隔 | 傾向 |
|------|------|
| 連闘・中 1 週 | 消耗リスク大 |
| 中 2-4 週 | 標準的なローテーション |
| 中 8-12 週 | やや間隔が開く。フレッシュさがプラス |
| 25-29 週（若馬） | 2-3 歳では回収率 238% という報告あり |

- **若馬（2-3 歳）の長期休養明け**: 成長して復帰するため期待値が非常に高い
- **馬体重 +20kg 以上（若馬）**: 成長の証拠で好走傾向
- **古馬の長期休養明け**: 明確な優位性なし

## 9. 季節性と開催パターン

| 季節 | 期間 | 主な特徴 |
|------|------|----------|
| 春 | 3-6 月 | クラシック路線（桜花賞→皐月賞→オークス→ダービー）。3 歳限定 G1 集中 |
| 夏 | 7-8 月 | ローカル場中心。主力級休養。2 歳新馬戦増加。荒れやすい |
| 秋 | 9-12 月 | 古馬 G1 集中（天皇賞秋・JC・有馬記念等）。最もレベルが高い時期 |
| 冬 | 1-2 月 | フェブラリーS（ダートG1）以外は準重賞以下が中心 |

**AI 構築への含意**: 季節をカテゴリ特徴量として投入。夏競馬はモデル精度が落ちやすい（下位層同士の対戦で差がつきにくい）。

## 10. 出走取消・競走中止・失格

| 事象 | 馬券 | データ処理 |
|------|------|-----------|
| 出走取消 | 該当馬番の馬券は全額返還 | `entry_status` で管理。特徴量計算・バックテストから除外 |
| 競走中止 | 返還なし | 着順なしとして処理 |
| 失格 | 確定着順に基づく（繰り上がり） | 確定払い戻しデータ（HJC）を使えば自動反映 |
| 降着 | 着順変更後の確定着順に基づく | 同上 |

**注意**: 前走が出走取消だった場合、ローテーション間隔の計算に影響する。

## 11. 馬具・装具

JRA では馬具の変更が走りに影響を与えることがある。JRDB の KYI データに蹄コード・馬具コードが含まれる。

| 馬具 | 効果 | AI での扱い |
|------|------|------------|
| ブリンカー | 視野制限で集中力向上 | 着用開始時に変化が出やすい。変更フラグが有効 |
| チークピーシーズ | 軽度の視野制限 | 同上 |
| シャドーロール | 下方の影への驚きを防止 | |
| バンテージ | 脚部保護 | |
| ホライゾネット | 頭部の視野制限 | |

**特徴量**: 「前走からの馬具変更フラグ」が有効。JRDB コード表 `bagu_code.txt` を参照。

## 12. 外国馬・短期免許騎手

- **外国馬**: G1 への海外遠征馬は国内 DB に過去走データがない。コールドスタート問題の変種
- **短期免許騎手**: 海外トップ騎手（C. ルメール、R. ムーア等）が参戦。国内成績のみで騎手特徴量を計算するため、短期免許初期は不完全
- **初期方針**: 外国馬はスキップし、Phase 5 で対応を検討

## 13. 情報公開タイムライン

予測システムの「いつ何のデータを使えるか」の整理。リーク防止の基盤。

| タイミング | 公開情報 |
|-----------|---------|
| 木曜 14-16 時 | 出走予定馬一覧（枠番未確定） |
| 金曜 9-10 時 | 枠順確定（一部 G1 は木曜 14 時に確定） |
| 金土 19 時 | JRDB 前日情報（KYI: IDM・各指数、KKA 等） |
| レース当日 発走約 60 分前 | 馬体重発表 |
| 土日 17 時 | JRDB 直前情報（TYB: パドック情報等） |
| 発売開始〜発走 | オッズがリアルタイム更新 |

**予測タイミング**:
- **前日予測（推奨・初期）**: 金曜夜に JRDB データ取得後、枠順・各指数込みで予測。馬体重・当日オッズは使えない
- **当日直前予測**: 馬体重発表後。TYB データ込み。精度は高いが運用が複雑

## 14. 芝→ダート転向（条件替わり）

条件替わりは馬券市場で最も歪みが生じやすい場面の一つ。

| パターン | 市場傾向 | AI の狙い |
|---------|---------|----------|
| 初ダート | 芝成績を過大評価しがち | 血統適性でダート向きなら過小評価の可能性 |
| 初芝 | 過小評価されやすい | 血統が芝向きなら穴馬候補 |
| 距離変更 | 前走距離の印象が強い | 血統・過去走の距離別成績で判断 |

**特徴量**: 転向フラグ、初挑戦フラグ、距離延長 / 短縮フラグ。Phase 2b で早期実装。

## 15. 競馬場の改修履歴

| 競馬場 | 改修期間 | 影響 |
|--------|---------|------|
| 京都 | 2020-2023 | 改修後に馬場傾向が大きく変化。改修前データの有効性に注意 |
| 中京 | 2012 | 改修済み。現在のデータはすべて改修後 |

**対策**: 基準タイム・馬場指数の算出は改修前後で分離する。条件別適性も改修後データのみで計算するのが安全。
